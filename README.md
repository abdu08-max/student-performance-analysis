# Student Performance Analysis

A simple data analysis project built using Python, Pandas, and Matplotlib.

## Features
- Calculate average marks
- Generate student rankings
- Identify toppers
- Visualize performance using bar charts

## Tools Used
- Python
- Pandas
- Matplotlib
